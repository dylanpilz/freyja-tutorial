# freyja-tutorial

Welcome to the Freyja analysis component of the amplicon sequencing workshop!

Use the left sidebar to browse files, and open them in the file viewer. Freyja should be pre-installed in your terminal. To verify, run `freyja --version`

You'll find the steps as directories in the left sidebar, starting with `Step1`. These contain all the instructions and data needed for each step of the tutorial. Go ahead and `cd` into `Step1` for the first step, and take a look at `Step1.md` by clicking on the file in the sidebar.

Have fun!
